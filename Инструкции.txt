КАК СОЗДАТЬ SVG путь для редактора

1.Сохранить макет медали в svg
2. Зайти в редактор svg (https://yqnn.github.io/svg-path-editor/)
3. Вставить пути из svg
4. Используя функции редактора уменьшить/увеличить весь макет, чтобы длина или ширина(в зависимости от того, что больше) стала равна 300.
5. По отдельности скопировать svg пути

КАК ДОБАВИТЬ SVG путь(картинку) для редактора

1. Скачать svg путь картинки
2. Зайти в редактор svg (https://yqnn.github.io/svg-path-editor/)
3. Вставить пути из svg
4. Используя функции редактора уменьшить/увеличить весь макет, чтобы длина или ширина(в зависимости от того, что больше) стала равна 100.
5. Сохранить путь
